- <h1>AdoWeather</h1>

    A weather application built for Android, providing weather forecasts based on your location or search queries.

    This app uses an external weather API to display current weather information, including temperature, humidity, wind speed, and other relevant data.

<br>

- <h1>Features</h1>

    Current Weather: Displays the current temperature, humidity, wind speed, and weather condition for the user's location.
    
    Location Search: Search for weather data by entering a city name.
    
    Forecast: View a detailed 5-day forecast.
    
    User-Friendly Interface: A simple and intuitive UI to help users quickly get weather information.

<br>

- <h1>Requirements</h1>

    Android Studio 4.0 or higher
  
    Minimum Android SDK version: 30

    API Key from [weatherapi.com](https://www.weatherapi.com/)

<br>

- <h1>Installation</h1>

  1. Clone the Repository
    To clone the repository to your local machine:
  
      git clone https://github.com/adonisthedev/AdoWeather.git
     
  2. Open the Project in Android Studio
     
      Open Android Studio and select Open an existing project.
  
      Navigate to the cloned project folder and open it.
  
  3. Set Up API Key

      Obtain an API key from [weatherapi.com](https://www.weatherapi.com/)
  
      Open app/src/main/res/values/strings.xml and add your API key:
  
      ```xml
      <string name="api_key">YOUR_API_KEY_HERE</string>

    
  4. Build the Project
     
      Click on Build > Make Project in Android Studio.
  
      If everything is set up correctly, you should be able to run the app on an emulator or a physical device.

<br>

- <h1>Usage</h1>
  
    Current Weather: When the app launches, it will automatically detect your location (if you have granted location permissions) and show the current weather.
  
    Search: Tap the search icon to search for weather data in any city.
